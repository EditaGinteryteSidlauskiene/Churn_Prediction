# Churn Prediction Dashboard (Streamlit)

Interactive dashboard for training, evaluating, and **explaining** churn models (Logistic Regression, Random Forest, XGBoost) on Telco/Internet datasets.  
Includes **SHAP**, **LIME**, and **Counterfactuals**.

## Features
- Model evaluation (AUC, recall, etc.)
- **SHAP** global & local explanations  
- **LIME** explanations  
- **Counterfactuals** (DiCE)
